- OpenDesign project: `ledgercalc-financial-calculator-design`
- Studio URL: http://127.0.0.1:7456/projects/ledgercalc-financial-calculator-design/conversations/c5a204e9-9a50-4fc7-bec4-b72333e64ff2/files/index.html

Notes:

- The artifact is a visual/design reference, not the production implementation.
- Production code lives in `src/` and keeps the finance engine tested in `src/lib/finance`.
- Use this artifact for future UI polish passes via Aider or Agor/OpenCode.
